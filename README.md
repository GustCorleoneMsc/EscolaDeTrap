# Escola para Rapper’s / Trappers — Boletim Digital

Este projeto consiste em uma landing page com estética de **boletim escolar**, desenvolvida como um exercício conceitual e visual. A proposta é simular um boletim anual aplicado a rappers e trappers, utilizando critérios artísticos como flow, drip, letra, atitude e originalidade.

O foco do projeto não é funcionar como um sistema real de avaliação, mas sim como uma peça visual e narrativa, misturando linguagem escolar tradicional com cultura urbana.

---

## Objetivo do Projeto

- Criar uma página estática com forte identidade visual
- Simular um boletim escolar de forma criativa
- Praticar boas práticas de HTML semântico e CSS organizado
- Manter o design simples, legível e com aparência institucional

---

## Tecnologias Utilizadas

- **HTML5**
  - Estrutura semântica (`header`, `main`, `footer`, `table`, `caption`, etc.)
  - Acessibilidade básica com uso correto de `scope`, `thead` e `tbody`

- **CSS3**
  - Estilo inspirado em boletins escolares impressos
  - Tipografia monoespaçada
  - Paleta restrita (azul, preto e vermelho para reprovação)
  - Layout responsivo com rolagem horizontal em dispositivos móveis

Não foram utilizados frameworks, bibliotecas externas ou pré-processadores.


---

## Responsividade

Em telas menores (como celulares), a tabela principal permanece legível através de rolagem horizontal. Essa abordagem foi escolhida para preservar a estrutura do boletim sem comprometer a leitura ou reduzir excessivamente o tamanho do texto.
